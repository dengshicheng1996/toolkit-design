### API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| type | 标签样式 | filling(填充)、filling-light(填充-浅)、icon（图标）、 line（线框）| filling |
| theme | 颜色主题 | normal 、orange、tangerine、green、| normal |
| size? | 大小 | 'sm' |  |
| onPress | 点击事件 | Event | - |
| className | 自定义类名 | string | - |
| component? | 可选type=icon时，有效 | React.ReactNode | - |
