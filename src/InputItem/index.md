### API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 受控 value | string | - |
| defaultValue | 非受控 defaultValue  | string | '' |
| placeholder | 占位 | string | - |
| onFocus | 获得焦点 | InputItemEvent | - |
| onBlur | 失去焦点  | InputItemEvent | - |

> 其他属性见 BaseFormItemType
